# LangChain Agent Lab

A simple open-source LangChain agent project with 3 tools:

- Calculator
- Knowledge (RAG over local notes)
- URL Summarizer

## Requirements

- Python 3.11
- An OpenAI API key

## Setup

### 1. Clone the repo

git clone https://github.com/YOUR-USERNAME/langchain-lab.git
